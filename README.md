# Agent Instructions

Setup guides and AI-agent install prompts for connecting tools to AI assistants
(Claude Desktop / Claude Code) for The Ark Church staff.

Each subfolder is a self-contained guide. **No secrets, tokens, or passwords are stored in this
repo** — credentials always stay on your own machine.

## Guides

| Guide | What it does |
|---|---|
| [**basecamp-mcp**](basecamp-mcp/README.md) | Connect **Basecamp** to your assistant via MCP — browse projects, manage todos, search, message boards, card tables, and more. Includes a [copy-paste agent install prompt](basecamp-mcp/AGENT_INSTALL.md). |
