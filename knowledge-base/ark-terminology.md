# The Ark Church — Terminology Reference

The correct names for everything at The Ark. When writing for The Ark, these terms are not suggestions — use them exactly. Full writing rules live in [ark-communications-style-guide.md](ark-communications-style-guide.md).

## Church name

| Context | Use |
|---------|-----|
| Formal (titles, signatures, locations) | The Ark Church |
| Casual reference | The Ark |
| As an adjective | Ark Church (e.g., "Ark Church communication") |
| Never | the ark church (lowercase) |

## Other terminology

| Term | Rule |
|------|------|
| attendee | not "attender" |
| worshiped, worshiper | not "worshipped, worshipper" |
| welcome (adjective) / welcomed (verb) | context-dependent |
| local mission partner | no "s" on mission; lowercase unless in a header |
| Team | Capitalize for a specific Ark ministry ("Communications Team"); lowercase generically ("volunteer teams") |
| Room names | "the Loft" not "The Loft"; "Room 101" not "room 101" |
| birth | use when referring to the youngest age served (not "newborn") |
| t-shirt | lowercase "t" (not "tshirt", "tee shirt") |
| y'all | acceptable in written communications |
| Check-In (noun) / check in (verb) | — |
| sign-up (noun) / sign up (verb) | — |
| login (noun) / log in (verb) | — |
| email, website, online | one word, no hyphens |
